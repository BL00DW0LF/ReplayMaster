# ReplayMaster
A tool for anonymizing SpyParty .replay files and more.  It can read most data in the header of .replay files, and allows you to adjust some parameters, useful for event organization that requires anonymizing replays.  

Replays can be edited individually, or in bulk by dragging multiple files at once, which are read in the order the OS provides them (usually the file clicked and dragged first, then the rest in sorted order).  If enabled, Bulk Mode helps by autofilling pseudonyms for each original player name, which lets you maintain consistent names per participant.  This data is not stored, so it is lost on refresh or when the page is closed.  

As noted on the page, these modifications are destructive, so they should only be performed on copies.

For use of ReplayMaster, [it is currently hosted on bl00dw0lf.com](https://bl00dw0lf.com/SpyParty/ReplayMaster/).
